# 運転代行シフト作成ルール（2026年1月版・確定）

## ① 営業・基本条件

- **営業時間**：20:00〜翌6:00
- **車両**：2台（1号車・2号車）
- **1台につき【2種免許＋1種免許】または【2種免許＋2種免許】のペア**
- **1種×1種は不可、2種×2種は可**
- **ピーク時間**：23:00〜翌2:00
- **月曜日は定休日**（全員休み）

## ② 最重要原則（絶対ルール）

- **全員、提出した「希望外の日・希望外の時間帯」は絶対に入れない**
- **希望に書かれていない日は**
  → 「出勤不可」として扱う
- **希望に書かれていない時間帯は**
  → その時間は稼働不可

## ③ メンバー別ルール（免許＋制約）

### 🔵 西村（2種）

- **基本**：23時以降稼働
- **金曜・土曜・1月2日は20:00から稼働可能**
- **1月3日は23時から稼働可能**
- **日曜**：原則休み
  - ただし「誰もいない場合」は例外的に出勤可
- **1月1日**：休み

### 🟢 鈴木（2種）

- **稼働可能日は本人提出分のみ**
- **9・10日**：20:00〜ラスト
- **13〜16日**：0:30〜ラスト
- **17日**：20:00〜ラスト
- **23・24日**：20:00〜ラスト
- **27〜30日**：0:30〜ラスト
- **31日**：20:00〜ラスト
- **上記以外の日・時間は一切不可**

### 🟡 チョロモン（2種）

- **23時以降のみ**（火曜日は20時から稼働可能）
- **他に2種がいない場合のみ稼働**
- **常用しない**（穴埋め専用）

### 🟠 たかし（1種）

- **木曜日は休み**
- **希望が出ている日・時間のみ稼働可**
- **2種がいない日は単独不可**（1種×1種禁止）

### 🔴 なみ（1種）

- **日曜・月曜は休み**
- **基本**：23時以降〜ラスト
- **例外的な希望（記載日）**
  - **2日**：20時〜可
  - **5〜8日**：21時〜2時
  - **11日**：19時〜ラスト
  - **12〜14日**：21時〜2時
  - **20〜22日**：21時〜2時
  - **23・24日**：2:30頃〜（早ければ0時〜）
  - **28・29日**：21時〜1時
- **上記以外は不可**

## ④ 金曜・土曜の運用ルール

- **金曜・土曜は売上最優先**
- **可能な限り**
  - **20:00〜**：1台確保
  - **23:00以降**：2台稼働を目指す
- **ただし**
  - **希望厳守を最優先し、無理な2台化はしない**

## ⑤ 例外ルール（明文化）

- **「誰もいない場合は西村が出勤」**
  - ただし
  - **1種免許者が誰もいない場合は不可**（1台不成立）
- **休業日は**
  - **「希望厳守の結果、1台も成立しない日」と定義**

## ⑥ シフト作成時のチェックリスト（運用用）

作成後、必ず以下を確認する：

- [ ] 希望外の日・時間が入っていない
- [ ] 全シフトが「2種＋1種」または「2種＋2種」になっている
- [ ] 1種×1種が存在しない
- [ ] 木曜にたかしが入っていない
- [ ] 日曜に西村が入っている場合は「例外扱い」
- [ ] チョロモンが常用されていない
