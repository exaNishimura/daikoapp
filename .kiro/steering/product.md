# Product Overview

代行運転における配車管理Webアプリケーション（MVP）。電話で受けた依頼を配車係が手入力し、車両タイムラインにドラッグ&ドロップで割り当て・確定できる機能を提供する。

## Core Capabilities

- **電話依頼登録**: 配車係による手入力による依頼作成
- **視覚的タイムライン管理**: 車両ごとのタイムライン表示とドラッグ&ドロップによる配車割り当て
- **自動ルート計算**: Google Maps APIを使用した所要時間の自動計算
- **競合検出**: 同一車両での時間重複を検出し、配車ミスを防止

## Target Use Cases

- 鈴鹿市近辺の代行運転における配車オペレーション
- 電話受注から配車確定までの一連の業務フロー
- 複数車両の配車状況を視覚的に管理

## Value Proposition

配車オペレーションを1画面で完結させ、視覚的なタイムライン管理と自動化されたルート計算により、配車効率を向上させる。MVPでは配車機能に特化し、会員管理・決済・ドライバーアプリ等は後回し。
